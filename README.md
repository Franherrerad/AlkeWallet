# Alke Wallet

Proyecto Frontend desarrollado como parte del módulo
**Trabajo práctico M2: Fundamentos del Desarrollo Frontend**.

## Descripción
Alke Wallet es una billetera digital que permite:
- Iniciar sesión
- Visualizar saldo disponible
- Realizar depósitos
- Enviar dinero a contactos
- Ver historial de transacciones

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- jQuery
- SweetAlert2
- LocalStorage

## Estructura del proyecto
El proyecto está organizado por carpetas para HTML, CSS y JavaScript,
siguiendo buenas prácticas de desarrollo frontend.

## Cómo ejecutar el proyecto
1. Clonar el repositorio
2. Abrir `login.html` en el navegador
3. Usuario de prueba:
   - Email: usuario@wallet.com
   - Contraseña: 1234
